# Demo Box

## Parâmetros Configuráveis
| Parâmetro | Tipo | Padrão | Descrição |
|-----------|------|--------|-----------|
| width | number | 50 | Box Width (mm) |
| depth | number | 50 | Box Depth (mm) |
| height | number | 30 | Box Height (mm) |
| wallThickness | number | 2 | Wall Thickness (mm) |
| cornerRadius | number | 3 | Corner Radius (mm) |
